Español:
Solo se usa Express.js de versión 4.19.2. Se usa npm (se necesita Node.js ya instalado). Luego:
- Abrir Command Prompt, ir al directorio de Express
- Ejecutar este comando para instalar los paquetes necesarios (en este caso, solo Express)
	npm install
- Ejecutar este comando para iniciar el servicio
	node app.js
- El servicio estará en esta URL:
	http://localhost:3000/
Si ya se ocupada el puerto 3000, podemos cambiar a otro puerto modificando la primera línea del archivo app.js También necesitamos actualizar un poco el código C#.
